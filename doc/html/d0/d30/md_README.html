Quadratic root finder

Almost the single most overkill version of a quadratic root finder I could think of. I could also add the quadratic functions to a polynomial struct with function pointers (i.e., as methods) to make this unnecessarily object oriented, but that might be best left to the user as an exercise.

You will note that I include Doxygen-style documentation and it is tooled for the usual gcc/cmake workflow. It's aching to use github actions as a build system.

I made this as a reference program to highlight a bunch of common C ideas. There are preprocessor macros, static functions, standard librarys, command line arguments, etc. etc.

Build Instructions

Since this is intended as a simple reference for undergraduate students, I intended this to be built and used in your favorite *nix flavor. These instructions will cover installing, building, and running the program in Debian on Windows via WSL. You could achieve the same in Windows with mingw, or even easier to do it on a native Linux device or macOS.

Installing WSL if you are using Windows

Consult the official install docs if you run into trouble.

First, fire open Command Prompt (search for it in your Start menu) or PowerShell, or even better, Terminal, and install debian via wsl

wsl --install -d Debian

Alternatively, try out ubuntu if you prefer the bleeding edge

wsl --install -d Debian

Installing the build tools and such

Assuming you are staring at a Debian-based Linux prompt, you should probably get up to date

sudo apt update
sudo apt upgrade -y

and now install all the magic sauce

sudo install git cmake vim

If you like Visual Studio Code instead of vim (heathen) then you can install on native Debian-based via snap with

sudo snap install --classic code

or follow the instructions to install via apt (I prefer it this way). If you are using WSL, there are instructions for that too.

Building the code

Now that you are all set up, and assuming you have generated an SSH key and added it to your github account, first clone the repo

git clone git@github.com:jacobw56/quadratic_formula.git

then make a build folder and build it

cd quadratic_formula
mkdir build
cd build
cmake ..
make

Now try it out!

./quadratic_formula -A=2.34 -B=7 -C=-4.51

or

./quadratic_formula --help
